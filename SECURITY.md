# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch state for this portfolio project.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive findings.

Report privately with:
- affected component/file
- reproduction steps
- impact assessment
- suggested fix (optional)

If this is your own fork, use your preferred private channel.

## Scope Notes

- Never commit credentials or private keys.
- `.env` is ignored; only `.env.example` should be committed.
- Use `python3 scripts/final_audit.py` before publishing.
